This package provides Twilio's .eslintrc as an extensible shared config.

## Installation

```bash
$ npm install eslint-config-twilio --save-dev
```

## Usage

Add an ESLint config to your `package.json`:

```json
{
	"name": "my-project",
	"eslintConfig": {
		"extends": "eslint-config-twilio"
	}
}
```

Or to `.eslintrc`:

```json
{
	"extends": "eslint-config-twilio"
}
```
