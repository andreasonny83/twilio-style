Twilio's approach to React styling.

## Installation

```bash
$ npm install eslint eslint-plugin-react eslint-config-twilio-react --save-dev
```

## Usage

Add the ESLint config to either your `package.json` or your `.eslintrc`:

### package.json

```json
{
  "name": "my-project",
  "eslintConfig": {
    "extends": [
      "twilio-react"
    ]
  }
}
```

### .eslintrc

```json
{
  "extends": [
    "twilio-react"
  ]
}
```
